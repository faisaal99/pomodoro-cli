# `pomodoro-cli`

An easy-to-use pomodoro timer, right in the terminal!


## Building

You need `cmake` installed.

```bash
mkdir pomodoro-cli
cd pomodoro-cli
mkdir build
git clone https://github.com/faisaal99/pomodoro-cli source
cmake -S source -B build
cmake --build build
```
